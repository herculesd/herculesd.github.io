No Javascript Dark Mode Toggle

Sometimes javascript fails, and sometimes people have it switched off.

For those people it's still nice to have the option of toggling dark-mode on and off.

This example uses:

  • No javascript
  • media-queries to default to user preferences.
  • css variables to make styling way easier.

If you're interested in learning how to make this you can read the tutorial here: endtimes.dev

or here: dev.to

See a live version using five color schemes and radio buttons here: missingdice.com
